## 📌 Getting Started

To run this project locally:

1. Clone the repository

   ```
   git clone https://github.com/neelimachallagundla/dev-resources.git
   ```

2. Navigate to the project folder

   ```
   cd dev-resources
   ```

3. Open `index.html` in your browser
